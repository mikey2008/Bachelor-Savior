# 🧑‍🍳 Bachelor Savior

Bachelor Savior is a smart and magical recipe web application that helps you cook amazing meals using whatever ingredients you have in your fridge! Built with HTML, CSS, JavaScript, and powered by the Gemini AI.

## Features ✨
- **Ingredient-Based Inspiration**: Generate comprehensive recipes based purely on the ingredients you currently have.
- **Cuisine Selection**: Filter generation by cuisine types like Italian, Mexican, Indian, Japanese, and more.
- **Dietary Restrictions**: Easily apply custom dietary filters (e.g., Vegan, Gluten-Free) to ensure the meal fits your needs.
- **Recipe Book**: A beautiful, interactive 3D UI to store and view your saved secret recipes locally.
- **Local Storage**: All your saved recipes and settings are stored locally on your device for fast access and privacy.

## Setup and Usage 🚀
1.  **Clone or download** this repository.
2.  **Backend Configuration**:
    -   Go to the `backend` folder.
    -   Copy `.env.example` to `.env`.
    -   Set your `GEMINI_API_KEY` and other secrets.
    -   Run `npm install` and `npm start`.
3.  **Frontend Access**:
    -   Open `index.html` via a local server (e.g., VS Code Live Server).
    -   The app will automatically proxy AI requests through your backend.
4.  **Start Cooking!** 😋


## Technologies Used 💻
- **HTML5** & **Vanilla CSS3** (Custom properties, Flexbox, CSS Grid)
- **Vanilla JavaScript** (ES6+)
- **Gemini API** for AI-powered recipe generation
- **Marked.js** to render markdown text within the browser
